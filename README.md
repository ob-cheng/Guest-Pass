# Guest Pass

A simple, client-side WiFi QR code generator.

## How to use

Since this is a static website, you don't need to install anything!

**Option 1: Direct Open**
- Double-click `index.html` to open it in your browser.

**Option 2: VS Code Live Server**
- If you use VS Code, right-click `index.html` and select **"Open with Live Server"**.

## Features
- purely client-side (no data leaves your device)
- Generates WiFi QR codes
- Print-friendly mode
- "Share Image" feature for mobile
